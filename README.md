# plant_tracker

Plant Care & Growth Tracker is a smart and user-friendly mobile application built with Flutter that helps users monitor and manage their plants efficiently. The app allows users to track plant growth, watering schedules, sunlight exposure, and overall plant health in one place. With reminders, progress tracking, and an intuitive interface, it makes plant care simple and organized for both beginners and gardening enthusiasts.
